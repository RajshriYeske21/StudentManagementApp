# Student Management App

The Student Management App is a Java-based application for managing student records. It allows you to add and remove students and list all the students in your database.

## Features

- **Add Student**: You can add students by providing their roll number, name, and grade.
- **Remove Student**: You can remove students by specifying their roll number.
- **List All Students**: The application provides a list of all the students in the database.
- **Simple Interface**: The user interface is text-based and easy to use.
